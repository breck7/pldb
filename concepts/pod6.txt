Pod6 is an easy-to-use markup language. It can be used for writing language documentation, for documenting programs and modules, as well as for other types of document composition

- Tags: text markup language
- Pod6 is developed on GitHub and has 9 stars
- Early development of Pod6 happened in https://github.com/zag/js-pod6/issues
- Pod6 is written in TypeScript, JavaScript, JSON, Markdown, YAML
- pod6.in was registered in 2012

=begin pod
 This is an ordinary paragraph

    While this is not
    This is a code block
    
    =head1 Mumble mumble
    
    Suprisingly, this is not a code block
        (with fancy indentation too)

But this is just a text. Again

=end pod
