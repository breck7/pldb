Multi-user BASIC was a dialect of the BASIC language for the DEC PDP-11 running the RT-11 operating system.  One or more users were supported in separate address spaces sharing the same language interpreter.  The syntax of the language was similar to but not identical to BASIC-11. Read more on Wikipedia...
