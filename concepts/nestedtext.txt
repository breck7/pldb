Human Readable and Writable Data Interchange Format

- NestedText website
 https://nestedtext.org
- NestedText is developed on GitHub and has 270 stars
- NestedText first developed in https://nurdletech.com
- NestedText is written in Python, reStructuredText, YAML, Ini, TOML, JSON, Make, XML, Bourne shell, CSV, CSS
- Pygments supports syntax highlighting for NestedText
