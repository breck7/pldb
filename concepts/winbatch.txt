Winbatch is a Microsoft Windows scripting language originally developed by Wilson WindowWare and currently supported, maintained and enhanced by Island Lake Consulting LLC. Its environment includes an interpreter and a code editor along with a dialog designer and optional compiler to create self-contained executables. Its language structure and syntax is a cross between DOS batch command, Basic, Fortran, and C. Read more on Wikipedia...
