BARE is an interface design language created in 2020.

#2240on PLDB 4Years Old
Homepage

BARE is a simple binary representation for structured application data.

  • Tags: interface design language
  • Early development of BARE happened in Internet Engineering Task Force
  • Pygments supports syntax highlighting for BARE
  • baremessages.org was registered in 2020
