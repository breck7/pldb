Flatline is a lispy language for the specification of values to be extracted or generated from an input dataset, using a finite sliding window of input rows.

- Tags: programming language
- Flatline is developed on GitHub and has 26 stars
- Early development of Flatline happened in BigML Inc
- Flatline is written in Python, reStructuredText, JavaScript, Markdown, HTML, Make, Jupyter Notebook, CSS
