GeneXus is a Cross-Platform, knowledge representation-based, development tool, mainly oriented to enterprise-class applications for  Web applications, smart devices and the Microsoft Windows platform. A developer describes an application in a high-level, mostly declarative language, from which native code is generated for multiple environments. It includes a normalization module, which creates and maintains an optimal database structure based on the user views of the reality described in a declarative (rule-based) language.
