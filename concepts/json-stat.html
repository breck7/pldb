JSON-stat is a simple lightweight JSON dissemination format best suited for data visualization, mobile apps or open data initiatives, that has been designed for all kinds of disseminators.

  • Tags: jsonFormat
  • JSON-stat is developed on GitHub and has 22 stars
  • Early development of JSON-stat happened in https://github.com/jsonstat
  • JSON-stat is written in Markdown
  • json-stat.org was registered in 2011
