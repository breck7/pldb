Write visualisation using a simple declarative markup like you would write code. Just wrap it in fenced block (three backticks) and mark the language as `vis`.

- Tags: text markup language
- visdown is developed on GitHub and has 663 stars
- visdown is written in JavaScript, Markdown, SVG, CSV, JSON, CSS, HTML
- visdown.com was registered in 2017

```vis
data:
  url: data/cars.csv
mark: point
encoding:
  x:
    field: kmpl
    type: quantitative
  y:
    field: price
    type: quantitative
 ```
