Hjson
=====

Hjson is an open source data notation created in 2014 by Christian Zangl.

Hjson is a syntax extension to JSON. It's NOT a proposal to replace JSON or to incorporate it into the JSON spec itself. It's intended to be used like a user interface for humans, to read and edit before passing the JSON data to the machine.

- Tags: dataNotation
- Hjson is developed on GitHub and has 2,647 stars
- Early development of Hjson happened in https://github.com/hjson
- Hjson is written in JSON, HTML, Markdown, JavaScript, CSS, XML, SVG, Bash
- hjson.org was registered in 2015
