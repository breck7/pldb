Sieve is a programming language that can be used for email filtering. It owes its creation to the CMU Cyrus Project, creators of Cyrus IMAP server. The language is not tied to any particular operating system or mail architecture. Read more on Wikipedia...

# Sieve filter

# Declare the extensions used by this script.
#
require [&quot;fileinto&quot;, &quot;reject&quot;];

# Messages bigger than 100K will be rejected with an error message
#
if size :over 100K {
   reject &quot;I&#39;m sorry, I do not accept mail over 100kb in size.
Please upload larger files to a server and send me a link.
Thanks.&quot;;
}

# Mails from a mailing list will be put into the folder &quot;mailinglist&quot;
#
elsif address :is [&quot;From&quot;, &quot;To&quot;] &quot;mailinglist@blafasel.invalid&quot; {
   fileinto &quot;INBOX.mailinglist&quot;;
}

# Spam Rule: Message does not contain my address in To, CC or Bcc
# header, or subject is something with &quot;money&quot; or &quot;Viagra&quot;.
#
elsif anyof (not address :all :contains [&quot;To&quot;, &quot;Cc&quot;, &quot;Bcc&quot;] &quot;me@blafasel.invalid&quot;,
header :matches &quot;Subject&quot; [&quot;*money*&quot;,&quot;*Viagra*&quot;]) {
      fileinto &quot;INBOX.spam&quot;;
}

# Keep the rest.
# This is not necessary because there is a &quot;implicit keep&quot; Rule
#
else {
     keep;
}
