Fenix Project is the codename for a GNU project to create a free compiler for a scripting language derived from the one created by Hammer Technologies for the game development suite DIV Games Studio. However, several features have been added which make it incompatible with most games programmed with DIV Games Studio.. Read more on Wikipedia...
