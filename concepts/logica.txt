Logica is an open source declarative logic programming language for data manipulation. Logica is a successor to Yedalog, a language created at Google earlier.

- Tags: queryLanguage
- Logica is developed on GitHub and has 1,710 stars
- Early development of Logica happened in https://github.com/EvgSkv/logica/issues
- Logica compiles to SQL
- Logica is written in Lex, Python, Jupyter Notebook, Markdown, JSON, HTML, YAML, JavaScript, Vim script, Protocol Buffers, CSS, CSV
- logica.dev was registered in 2020

# Define natural numbers from 1 to 29.
N(x) :- x in Range(30);
# Define primes.
Prime(prime: x) :-
  N(x),
  x &gt; 1,
  ~(
    N(y),
    y &gt; 1,
    y != x,
    Mod(x, y) == 0
  );
