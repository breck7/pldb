Implementation of the λΠ-calculus modulo rewriting

 https://deducteam.github.io
- dedukti is developed on GitHub and has 155 stars
- dedukti first developed in https://github.com/Deducteam
- dedukti is written in OCaml, Bourne shell, Markdown, Make, Lisp, XML, YAML, Vim script, Python

Nat: Type.
zero: Nat.
succ: Nat -&gt; Nat.
def plus: Nat -&gt; Nat -&gt; Nat.
[ n ] plus zero n --&gt; n
[ n ] plus n zero --&gt; n
[ n, m ] plus (succ n) m --&gt; succ (plus n m)
[ n, m ] plus n (succ m) --&gt; succ (plus n m).
