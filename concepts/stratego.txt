Stratego/XT is a language and toolset for constructing stand-alone program transformation systems.  It combines the Stratego transformation language with the XT toolset of transformation components, providing a framework for constructing stand-alone program transformation systems. The Stratego language is based on a programming paradigm called strategic term rewriting. Read more on Wikipedia...
