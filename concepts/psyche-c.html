Psyche-c is a compiler for incomplete C code. It features a Hindley/Milner-inspired type inference engine for C.

  • Tags: compiler
  • psyche-c is developed on GitHub and has 526 stars
  • Early development of psyche-c happened in https://github.com/ltcmelo/psychec/issues
  • psyche-c is written in C++, C, Python, CMake, Markdown, YAML, Haskell, Bourne shell, Pascal
