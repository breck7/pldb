Jsonnet - The data templating language

- Tags: dataNotation
- Jsonnet is developed on GitHub and has 6,778 stars
- There are at least 1,430 Jsonnet repos on GitHub
- Jsonnet is a superset of JSON
- Early development of Jsonnet happened in Google
- Jsonnet is written in JavaScript, HTML, C++, SVG, Bourne shell, Python, Markdown, Bazel, YAML, CMake, JSON, CSS, Java, C, Bash, starlark, XML, Make, Dockerfile
- The  Google BigQuery Public Dataset GitHub snapshot shows 104 users using Jsonnet in 190 repos on GitHub
- jsonnet.org was registered in 2015
