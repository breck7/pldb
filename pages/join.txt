Become an active contributor to PLDB
====================================

Do you seek programming language enlightenment?

Why not seek that joyfully with colleagues while serving others, making an impact, and accumulating internet points?

Become an active contributor to PLDB!

Tasks you could do
==================
- Write a data backed post on your blog and share it with us so we can link to it
- Add or edit languages to the ScrollSet
- Add or edit language features and examples to the ScrollSet
- Improve or add a data importer for a source like Wikipedia or Github
- Use or connect AI models to add data to the ScrollSet
- Discuss and improve the measures and ranking algorithm
- Improve the website or codebase

To get involved say hi on Twitter or send a pull request.
