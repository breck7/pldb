Line Comments
=============

// A comment

# A comment

-- A comment

% A comment

; A comment

* A comment

MOVE ABC to XYZ  *&gt; This is a comment.

&#39; A comment

C A comment

// This is a line comment in Gleam

-# A comment

\ A comment

⍝ A comment

NB. A comment

REM A comment

\* A comment

; I am a comment

/ A comment

! A comment

; a comment

-- Counts iterations.

__ A comment

// comment

--- A comment

~ A comment

K) A comment

#&lt;A comment&gt;

;; A comment

// The same as c

COMMENT A comment

DO END  ! Do nothing

“”” A comment
